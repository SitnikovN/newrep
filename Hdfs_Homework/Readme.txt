Для конвертация данных из csv в avro формат использовал библиотеку hdfs и ее расширение hdfs.ext.avro
ссылка на API:
https://hdfscli.readthedocs.io/en/latest/api.html#
Подключение создается непосредственно к web порту Namenod'ы (9870 в моем случае).

Исходные предположения о данных:
 в первых строчках файлов содержатся заголовки
 все типы данных для упаковки сообщения - string
 
Скрипт из указанной папки делает конвертацию всех файлов в Avro.
Далее простыми командами просто создал директорию и перекинул туда все файлики.
